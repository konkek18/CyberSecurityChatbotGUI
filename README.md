# Cybersecurity Awareness Chatbot GUI

## Features
- Windows Forms GUI with dark theme
- Voice greeting (plays Greeting.wav at startup)
- ASCII art displayed in chat area
- Keyword recognition: password, phishing, scam, privacy, 2FA
- Random responses using lists and dictionary
- Sentiment detection (worried, curious, frustrated)
- Memory: remembers user name and favorite topic
- Conversation flow: responds to "tell me more", "another tip"
- Error handling for unknown inputs

## How to Run
1. Open the solution in Visual Studio
2. Press F5 to build and run
3. Enter your name when prompted
4. Type cybersecurity questions or greetings

## Technologies
- C# .NET Windows Forms
- System.Media for audio playback
